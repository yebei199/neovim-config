# lua - Neovim Lua配置核心

本目录是整个Neovim配置的核心入口，用Lua编程语言编写。Lua是Neovim官方推荐的配置和扩展语言，相比VimScript提供了更好的可读性、更强的功能和更灵活的编程能力。

## 为什么使用Lua

Lua相比VimScript的优势：
- **易读易维护** - 清晰的语法结构，更接近现代编程语言
- **性能优越** - Lua比VimScript运行快得多，配置启动更迅速
- **功能完整** - 支持更丰富的数据结构和编程特性
- **社区繁荣** - Neovim社区逐步迁移至Lua，相关资源丰富

## 目录架构

本目录将配置分为三个主要模块：

### config/ - 配置与初始化
包含Neovim的核心配置、选项设置和初始化逻辑。涵盖：
- 自动命令(autocmds) - 定义事件处理规则
- 快捷键映射(keymaps) - 所有按键绑定
- 语言配置(langs) - 编程语言的工具链设置
- 选择器(pickers) - Telescope等选择器配置
- 高亮、图标、选项等基础设置

### plugins/ - 插件定义
存放所有Neovim插件的规范和配置。使用lazy.nvim作为插件管理器，按功能分类：
- component/ - UI组件和视觉增强
- editor/ - 编辑功能增强
- lsp/ - 语言服务器和智能补全
- ui/ - 界面和主题相关
- util/ - 工具类插件和辅助功能

### utils/ - 工具库和辅助模块
提供通用的工具函数和辅助模块，为配置和插件提供支持：
- 文件操作、列表处理、高亮管理等工具函数
- 语言特定的辅助工具
- 插件相关的辅助函数
- 通用的工具库

## 工作原理

Neovim启动时会自动加载 `init.lua`，该文件协调加载这三个模块，最终构成完整的编辑环境。这种模块化结构使得配置易于组织、维护和扩展。
