# util - 工具类插件与辅助功能

工具类插件是Neovim配置的核心组成部分，专注于提升开发效率和用户体验。这些插件提供任务管理、快速导航、通知优化等辅助功能，帮助开发者简化工作流，专注于核心任务。

本模块包含八个主要插件：

- **flatten**：简化窗口布局，支持嵌套窗口扁平化处理。
- **grapple**：快速跳转，通过标记系统实现文件间瞬时切换。
- **lazydev**：Neovim开发支持，懒加载机制优化性能。
- **noice**：通知系统重构，将命令行消息转换为现代化弹出通知。
- **overseer**：任务运行器，支持异步任务管理和进度监控。
- **snacks**：小工具集合，提供快速搜索、文件预览等功能。
- **whichkey**：键位提示插件，实时显示可用快捷键。
- **zk**：知识库工具，支持笔记管理和链接创建。

工具类插件具有模块化、可扩展的特点：独立工作但协同配合，轻量级高性能，避免资源浪费。

最佳实践：根据项目需求选择启用插件，避免过度配置；结合快捷键实现无缝集成；定期更新获取最新功能。这些插件显著提升生产力，使Neovim体验更加现代化。
