# editor - 编辑功能增强

本目录包含专注于增强代码编辑功能的插件配置。这些插件提供了高级的编辑操作，使得文本编辑更加高效和强大。

## 插件说明

### flash.lua - 快速导航
Flash提供了快速的光标导航功能，允许用户以最少的按键快速跳转到目标位置：
- 快速定位到可见范围内的任何字符或单词
- 可视化的标记提示，显示可达的位置
- 与vim的f/t命令风格一致

这个插件大幅减少了导航所需的按键次数。

### surround.lua - 配对符号处理
管理和编辑配对符号（括号、引号等）：
- 快速添加、修改、删除配对符号
- 支持嵌套的配对符号
- 支持自定义符号对
- 与vim习惯一致的操作方式

例如可以快速将 `word` 改成 `"word"` 或 `[word]`。

### mini-ai.lua - 智能文本对象
提供了增强的文本对象功能：
- 更智能的参数、函数体等代码块识别
- 支持自定义文本对象
- 改进了括号、引号内容的选择

这使得修改和删除代码块变得更加直观。

### pair.lua - 配对符号自动补全
在输入时自动配对输入的符号：
- 输入`(`时自动补全`)`
- 支持嵌套配对的智能处理
- 支持多种编程语言的符号规则

### gitsign.lua - Git差异展示
在行号列显示Git变更指示：
- 显示新增、修改、删除行
- 快速预览diff和还原修改
- 支持选择性的阶段提交

这个插件提供了直观的代码版本控制反馈。

### grug-far.lua - 高级搜索替换
提供了项目范围内的高级搜索和替换功能：
- 同时支持多个搜索模式
- 预览替换结果
- 高效的批量替换

### substitute.lua - 替换操作增强
增强Vim的替换操作(`s`和`r`)：
- 更直观的替换命令
- 支持Treesitter感知的替换
- 更好的模式匹配

### coerce.lua - 文本转换
提供了文本大小写和格式转换功能：
- 驼峰、蛇形、大写等格式转换
- 快速转换变量名和标识符格式

### yanky.lua - 复制历史管理
增强了Vim的复制/粘贴功能：
- 完整的复制历史管理
- 循环访问复制历史
- 复制和粘贴的更多灵活性

## 设计特点

这些编辑增强的共同特点：
- **直观性** - 操作方式与vim内建功能保持一致
- **强大性** - 提供比vim内建更多的功能和灵活性
- **可靠性** - 经过社区验证的稳定插件
- **互补性** - 插件间相互配合，协同增强编辑体验

## 最佳实践

使用这些编辑工具时的建议：
- 逐个学习每个插件的快捷键和用法
- 在实际编辑中多次实践以形成肌肉记忆
- 根据个人习惯自定义快捷键映射
- 避免过度复杂的配置，保持工作流的简洁
